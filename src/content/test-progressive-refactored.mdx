Let's start with a fundamental question: How do we mathematically describe all possible outcomes of an experiment?

<SectionBreak />

## What is a Sample Space?

In probability theory, we use a **sample space** to represent all possible outcomes.

<Definition term="Sample Space">
  A sample space $S$ is the set of all possible outcomes of a random experiment.
</Definition>

<SectionBreak />

## Examples

Let's look at some examples:

<BulletList color="blue">
  <li><strong>Coin flip:</strong> $S = \{\text{Heads}, \text{Tails}\}$</li>
  <li><strong>Die roll:</strong> $S = \{1, 2, 3, 4, 5, 6\}$</li>
  <li><strong>Two coin flips:</strong> $S = \{HH, HT, TH, TT\}$</li>
</BulletList>

<Alert type="info">
  Notice how the sample space depends on the experiment!
</Alert>

<SectionBreak />

## What are Events?

<Concept title="Events" icon="ðŸŽ¯">
  An **event** is any subset of the sample space. It represents a collection of outcomes we're interested in.
</Concept>

For example, when rolling a die:

<BulletList color="purple">
  <li>Event $A$: "Rolling an even number" = $\{2, 4, 6\}$</li>
  <li>Event $B$: "Rolling less than 3" = $\{1, 2\}$</li>
</BulletList>

<SectionBreak />

## Combining Events

We can combine events using set operations:

<ExampleBlock 
  color="blue"
  title="Union ($A \cup B$)"
  description="Either A or B occurs"
>
  Example: $\{2, 4, 6\} \cup \{1, 2\} = \{1, 2, 4, 6\}$
</ExampleBlock>

<ExampleBlock 
  color="green"
  title="Intersection ($A \cap B$)"
  description="Both A and B occur"
>
  Example: $\{2, 4, 6\} \cap \{1, 2\} = \{2\}$
</ExampleBlock>

<ExampleBlock 
  color="red"
  title="Complement ($A'$)"
  description="A does not occur"
>
  Example: $\{2, 4, 6\}' = \{1, 3, 5\}$
</ExampleBlock>

<SectionBreak />
