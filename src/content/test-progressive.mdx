## What is a Sample Space?

In probability theory, we use a **sample space** to represent all possible outcomes.

> **Definition:** A sample space $S$ is the set of all possible outcomes of a random experiment.

<SectionBreak />

## Examples

Let's look at some examples:

<ul className="space-y-3 my-4">
  <li className="flex items-start">
    <span className="text-blue-500 mr-3 mt-1">•</span>
    <span><strong>Coin flip:</strong> $S = \{\text{Heads}, \text{Tails}\}$</span>
  </li>
  <li className="flex items-start">
    <span className="text-blue-500 mr-3 mt-1">•</span>
    <span><strong>Die roll:</strong> $S = \{1, 2, 3, 4, 5, 6\}$</span>
  </li>
  <li className="flex items-start">
    <span className="text-blue-500 mr-3 mt-1">•</span>
    <span><strong>Two coin flips:</strong> $S = \{HH, HT, TH, TT\}$</span>
  </li>
</ul>

Notice how the sample space depends on the experiment!

<SectionBreak />

## What are Events?

An **event** is any subset of the sample space. It represents a collection of outcomes we're interested in.

For example, when rolling a die:

<ul className="space-y-2 my-4">
  <li className="flex items-start">
    <span className="text-purple-500 mr-3 mt-1">•</span>
    <span>Event $A$: "Rolling an even number" = $\{2, 4, 6\}$</span>
  </li>
  <li className="flex items-start">
    <span className="text-purple-500 mr-3 mt-1">•</span>
    <span>Event $B$: "Rolling less than 3" = $\{1, 2\}$</span>
  </li>
</ul>

<SectionBreak />

## Combining Events

We can combine events using set operations:

<div className="space-y-6 my-6">
  <div className="border-l-4 border-blue-500 pl-6">
    <h3 className="text-lg font-semibold text-white mb-2">Union ($A \cup B$)</h3>
    <p className="text-neutral-300 mb-2">Either A or B occurs</p>
    <p className="text-sm bg-neutral-900 rounded p-3 font-mono">
      Example: $\{2, 4, 6\} \cup \{1, 2\} = \{1, 2, 4, 6\}$
    </p>
  </div>
  
  <div className="border-l-4 border-green-500 pl-6">
    <h3 className="text-lg font-semibold text-white mb-2">Intersection ($A \cap B$)</h3>
    <p className="text-neutral-300 mb-2">Both A and B occur</p>
    <p className="text-sm bg-neutral-900 rounded p-3 font-mono">
      Example: $\{2, 4, 6\} \cap \{1, 2\} = \{2\}$
    </p>
  </div>
  
  <div className="border-l-4 border-red-500 pl-6">
    <h3 className="text-lg font-semibold text-white mb-2">Complement ($A'$)</h3>
    <p className="text-neutral-300 mb-2">A does not occur</p>
    <p className="text-sm bg-neutral-900 rounded p-3 font-mono">
      Example: $\{2, 4, 6\}' = \{1, 3, 5\}$
    </p>
  </div>
</div>
